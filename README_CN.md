# RoundTable · 圆桌游戏

Flutter 打造的移动端桌游平台，首发德州扑克，后续扩展狼人杀等派对游戏。

## 特性

### 德州扑克 (Texas Hold'em)
- **蒙特卡洛作弊面板**：聊天框输入 `wenwangdk` 实时计算胜率 / 听牌 / 关键出牌，基于 1500 次蒙特卡洛模拟
- **心智扑克协议**：蓝牙联机采用模幂交换加密实现无信任荷官发牌——任一设备均不知全牌
- **环形编号座位**：网易狼人杀式布局，奇数 N 上 1 + 两侧 (N−1)/2，偶数 N 两侧各 N/2
- **30 秒倒计时圆环**：绿→黄→红顺时针扫过，超时自动弃牌/过牌
- **预选动作**：非自己回合提前勾选盖牌/让牌/跟注/加注，轮到时立即执行
- **齿轮推杆加注滑轨**：自定义 `SliderComponentShape` + 暗金属面板 + 快选（1/2底池/底池/全押） + 精确阻尼感
- **持久牌桌**：座位管理 / 买入补码 / 0 筹码自动坐 out / 解散结算 `净 = 手上筹码 − 累计买入`
- **AI 机器人**：基于底池赔率与牌力估算的启发式决策引擎（**计划接入大模型 AI**）
- **动作回馈**：弃牌头像变灰 / 跟注筹码弹跳 / 加注金光脉冲 + 系统提示音
- **D/SB/BB 标识** + **ALL IN 标签** + **庄家顺时针轮转**
- **摊牌亮牌**：仅赢家亮底牌，按离庄距离逐一展示，合并公共牌显示最佳 5 张组合

### 聊天 & 语音
- 文字气泡 + 麦克风长按录音（语音播放进度条）

### 蓝牙联机（开发中）
- 分布式心智扑克协议已通过 `MockTransport` 双节点进程内验证
- `flutter_blue_plus` BLE 中心角色传输层（待外设 Peripheral 方案完成 P2P）

## 架构

```
lib/
├── main.dart
├── poker/            # 纯 Dart：牌/牌组/7选5手牌评估/下注边池引擎/蒙特卡洛胜率计算
├── crypto/           # 心智扑克（模幂交换加密）
├── session/          # 持久牌桌控制器 / 联机会话
├── ai/               # 机器人决策
├── net/              # 传输抽象（Mock/蓝牙）
├── games/            # 游戏注册表（轮播驱动）
└── ui/screens/       # 游戏轮播 / 大厅 / 牌桌 / 蓝牙
```

## 快速开始

```bash
flutter pub get
flutter run -d chrome   # 桌面调试（无蓝牙）
flutter run -d <设备>    # 真机
flutter test            # 26 个单元测试全部通过
```

## 作弊码

| 命令 | 功能 |
|------|------|
| `wenwangdk` | 开启胜率面板（不发送到聊天） |
| `wenwanggb` | 关闭胜率面板（不发送到聊天） |
